# Django Signals & ORM Project

This project implements Django Signals, ORM techniques, and caching for a messaging application.

## Features

- User notifications via signals
- Message edit history
- Threaded conversations
- Custom ORM managers
- View caching

## Setup

1. Activate virtual environment: `source venv/Scripts/activate`
2. Install dependencies: `pip install django`
3. Run migrations: `python manage.py migrate`
4. Create superuser: `python manage.py createsuperuser`
5. Run server: `python manage.py runserver`
